Aura | Aesthetic Engine Pro

Aura is a high-performance, browser-based design token generator. It allows designers and developers to create cohesive, accessible color palettes and typography pairings with professional harmony algorithms.

‚ú® Features

Harmony Modes: Choose between Analogous, Monochrome, Complementary, and Triadic algorithms.

Dynamic Saturation: Fine-tune the vibrancy of your palette using a real-time slider.

Typography Pairing: Select from 20 curated Google Fonts to preview your colors in a real-world layout.

WCAG Contrast Checker: Automatic contrast ratio calculation (AA standard) between your primary brand color and the background.

Smart Swatches:

Click to Copy: Instantly copy HEX values to your clipboard.

Locking: Protect specific colors from being changed during regeneration.

State Persistence: Saves your favorite palettes and font choices to local storage.

CSS Export: Generates ready-to-use CSS :root variables for your project.

Dark Mode Support: Fully responsive UI that respects system preferences and includes a manual toggle.
